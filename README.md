# Mines
This is a game where player has mines which produces coins and can be withdrawed to upgrade mines.  

Instructions-  
1. Get coins by withdrawing from mines which produces on time
2. with coins, you can convert to gems
3. with coins, you can upgrage multiplier in steps of 0.1
4. with gems, you can increase mines
5. you can also save and retrieve progress through notepad by entering same player name
6. multiple player data is saved in single notepad called "Saves.txt"
